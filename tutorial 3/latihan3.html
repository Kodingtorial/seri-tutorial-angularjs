Filter currency
Saya memiliki uang sebanyak {{ 20000 | currency}}
Saya memiliki uang sebanyak {{ 20000 | currency: 'Rp'}}

Filter date
Hari ini tanggal {{ '2017-02-22' | date:'fullDate' }}
Hari ini tanggal {{ '2017-02-22' | date:'medium' }}
Hari ini tanggal {{ '2017-02-22' | date:'dd-MMM-yyyy' }}
Hari ini tanggal {{ '2017-02-22' | date:'dd-MM-yyyy' }}

Filter lowercase
Nama saya {{'Iwan Kurniawan' | lowercase}}

Filter uppercase
Nama saya {{'Iwan Kurniawan' | uppercase}}

{{ {Nama: 'Iwan Kurniawan', Usia: 30} }}
